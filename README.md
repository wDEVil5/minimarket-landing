# 🛒 SumarketExpress — Landing Page

Landing page para un minimarket de barrio (negocio familiar). El objetivo es que los vecinos y clientes puedan conocer los productos que se comercializan, las ofertas, la ubicación y los horarios, además de contar con un formulario de contacto para solicitar cotizaciones.

Como estudiante de Ingeniería en Computación e Informática, desarrollé este proyecto para aplicar y reforzar mis conocimientos de desarrollo front-end, partiendo de la base de que un negocio con presencia web transmite mayor confianza y valor.

## 🔗 Demo en vivo

[Ver sitio en vivo](https://wdevil5.github.io/minimarket-landing/)

## 🛠️ Tecnologías

- **HTML5** (estructura semántica)
- **CSS3** (Flexbox, Grid, variables, diseño responsive, animaciones)
- **JavaScript** (interactividad y validación)
- **Git & GitHub** (control de versiones y despliegue)

## ✨ Características

- Diseño totalmente **responsive** (móvil, tablet y escritorio)
- **Menú hamburguesa** funcional para dispositivos móviles
- Sección de productos en cuadrícula adaptable
- **Carrusel infinito** de productos con pausa al pasar el mouse
- Bloque de ofertas destacadas
- Sección de ubicación y horarios
- **Formulario de cotización** con validación de campos
- Animaciones suaves de aparición al hacer scroll
- Navegación interna con desplazamiento suave

## 📚 Lo que aprendí

Durante el desarrollo de este proyecto adquirí y reforcé:

- Maquetado con **HTML semántico** y buenas prácticas de accesibilidad
- Estilos modernos con **CSS** (sistema de variables, Flexbox, Grid, media queries, `clamp()` y animaciones con `@keyframes`)
- Fundamentos de **JavaScript**: variables, funciones, eventos, condicionales, manipulación del DOM e Intersection Observer
- Uso de **Git**: configuración del entorno, commits con mensajes descriptivos y manejo de un historial ordenado
- Despliegue de un sitio en **GitHub Pages** con una URL pública

## 💻 Cómo ejecutarlo localmente

1. Clona o descarga este repositorio.
2. Ábrelo con un editor de código (por ejemplo, **VS Code**).
3. Abre el archivo `index.html` con la extensión **Live Server** (clic derecho → "Open with Live Server"), o simplemente ábrelo en tu navegador.

También puedes verlo directamente en la [demo en vivo](https://wdevil5.github.io/minimarket-landing/).
